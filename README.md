# Sema Examples

Example applications built with [Sema](https://withsema.com) — universal ingestion infrastructure for AI Agents.

## Examples

| Example | Description | Stack |
|---------|-------------|-------|
| [bug-reporting-agent](./bug-reporting-agent) | Email bug reports → Linear issues | Python, Flask |

## What is Sema?

Sema handles the upstream plumbing for AI agents: email parsing, file ingestion, sender verification, and webhook delivery. You focus on the agent logic.

## Getting Started

Each example is self-contained. Pick one, follow its README, and you'll be running in minutes.
